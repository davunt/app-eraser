Permissions

App Eraser requires some permissions in order to find and move files to trash.

To allow App Eraser to work properly please do the following:

  1. Go to 'System Preferences'
  2. Go to 'Security and Privacy'
  3. Go to the 'Privacy' tab
  4. Enable the following permissions:
    1. Enable 'Finder' access in the 'Automation' section. This allows App Eraser to move files to trash
    2. Enable the folders you want to allow App Eraser to search in 'Files and Folders'
